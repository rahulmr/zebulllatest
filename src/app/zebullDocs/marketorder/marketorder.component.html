Market Order

URL

< BASE URL > + v1/placeOrder/executePlaceOrder


Method

Post

Input Parameters

S.NO KEY VALUE
1 discqty Int Quantity
2 trading_symbol String Trading Symbol
3 exch String Exchange(NSE or BSE or NFO or MCX)
4 transtype String Transaction Type (BUY or SELL)
5 ret String Retention Type(DAY or IOC)
6 prctyp String Price Type(L or MKT)
7 qty Int Quantity
8 price Int Price
9 pCode stirng Product Code(MIS)
10 symbol_id stirng Token
11 complexty stirng Order Type (REGULAR or CO or AMO)
12 trigPrice Int Price
