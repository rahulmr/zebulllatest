Validate 2FA

URL

../MobullService/v1/customer/validAnswer


Method

Post

Input Parameters

S.NO KEY VALUE
1 userId string User Id
2 sCount string Question count
3 sIndex string index
4 answer1 string answer for the question
5 answer2 string answer for the question

Example

           {{'{'}}
            "userId":"XY7656"
            "sCount":"2", 
            "sIndex": "27|25",
            "answer1": "abc",
            "answer2": "xyz" 
            {{'}'}}

Success Response

S.NO PARAMETERS DESCRIPTION
1 stat string Ok
2 message string Expiry Message
3 tdata string Ok
4 userSettingDto string JSON object

UserSettingDto JSON object contains below parameters
S.NO PARAMETERS DESCRIPTION
1 exch Enabled exchange in a JSON array
2 pCode Enabled order type in a JSON array
3 prctyp Enabled product type in a JSON array
4 s_prdt_ali Product alias
5 sTransFlg YES/NO
YES → both login password and transaction password is enabled in application
NO → only login password is enabled in application
6 default market_watch_name Default market watch name
7 broker_name Broker name of user
8 branch_id Branch id of user
9 market_watch_count Market watch count ( Only fetched from property file or product level , not user level)
10 email Email id of user
11 weblink It is a JSON array giving information about third party link
12 account_id Account id of user
13 exchDeatil JSON object
14 lots_weight Lots weight configurable flag
15 password special_character Password Special character flag
16 accountName Account name
17 userPrivileges User privilages
18 ySXorderEntry Flag to display YSX exchange button enable/disable (YES/NO)
18 criteriaAttribute_array JSONArray containing Set order Criteria flag (place, modify, cancel)

Error

S.NO PARAMETERS DESCRIPTION
1 start string Not_Ok
2 emsg string 1) Session Expired
2) Not able to Retrieve Valid Ans
3) User Blocked Contact System Administrator
4) No Licence
5) Permission Denied
