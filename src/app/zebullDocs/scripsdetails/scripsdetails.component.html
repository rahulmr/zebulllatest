Scrips Details

URL

< BASE URL > + v1/ScripDetails/getScripQuoteDetails


Method

Post

Input Parameters

S.NO KEY VALUE
1 exch String Exchange Name
2 symbol String Scrip Token

          {{'{'}}
            "exch": "nse",
            "symbol": "212"
          {{'}'}}
