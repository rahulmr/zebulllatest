Bracket Order

URL

< BASE URL > + v1/placeOrder/executePlaceOrder


Method

Post


Input Parameters

S.NO KEY VALUE
1 discqty Int Quantity
2 trading_symbol String Trading Symbol
3 exch String Exchange (NSE or BSE or NFO or MCX)
4 transtype String Transaction Type (BUY or SELL)
5 ret String Retention type (DAY or IOC)
6 prctype String Price type (L or MKT)
7 qty Int Quantity
8 price Int Price
9 pCode Int Product Code (MIS)
10 symbol_id String Token
11 target String Target Price
12 stopLoss String Stop Loss Price
13 trailing_stop_loss String Price
14 complexty String BO
15 trigPrice Int Trig Price

Success Response

S.NO PARAMETERS DESCRIPTION
1 stat String ok
2 nestOrderNumber Int Nest order number

Error

S.NO PARAMETERS DESCRIPTION
1 stat String Not_Ok
2 emsg String
  • Session Expired

  • Not able to Retrieve BO PlaceOrder

  • Error Occurred : < reason>

  • Trailing Tick should be multiple of tick size : [tickValue in rupees]
