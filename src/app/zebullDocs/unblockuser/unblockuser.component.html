Unblock User

URL

../MobullService/v1/customer/unblockUser


Method

Post

Input Parameters

S.NO KEY VALUE
1 userId string User Id
2 email string E-mail id
3 pan string PAN Number

Example

     {{'{'}} 
       "email": "abc@gmail.com",
        "pan": "AAAAA1111B"
    {{'}'}}

Success Response

S.NO PARAMETERS DESCRIPTION
1 stat string Ok
2 Result string User unblocked successfully

Error

S.NO PARAMETERS DESCRIPTION
1 stat string Not_Ok
2 emsg string 1. Not able to Retrieve Unblock User
2. Invalid User
3. Email ID does not match
4. PAN number does not match
5. Failed to unblock user. Kindly try again later
6.You aren't allowed to place a Password Request.
7.The Client details is not available
