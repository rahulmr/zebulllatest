Delete Scrips

URL

< BASE URL> + v1/marketWatch/deleteMWScrip


Method

Post

Input Parameters

S.NO KEY VALUE
1 mwName String Market Watch Name
2 exch String Exchange
3 symbol String Symbol Token

Example

         {{'{'}}
           "mwName": "abc",
           "exch": "NSE",
           "symbol": "46801"
         {{'}'}}

Success Response

S.NO PARAMETERS DESCRIPTION
1 stat String Ok
2 Result String success

Error

S.NO PARAMETERS DESCRIPTION
1 stat String Not_Ok
2 emsg String

  • Not able to Retrieve DeleteMWScrips

  • token is Not A Proper Symbol

  • Cannot Delete the Last Scrip
