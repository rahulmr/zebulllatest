Place Order

URL

< BASE URL > + v1/placeOrder/executePlaceOrder


Method

Post


Input Parameters

S.NO KEY VALUE
1 discqty int Quantity
2 trading_symbol String Trading Symbol
3 exch String Exchange(NSE or BSE or NFO or MCX)
4 transtype String Transaction Type (BUY or SELL)
5 ret String Retention type(DAY or IOC)
6 prctyp String Price type(L or MKT or SL or SL-M)
7 qty int Quantity
8 symbol_id String Token
9 price int Price
10 trigPrice int Trigger price
11 pCode String Product Code(MIS or NRML )
12 complexty String Order Type (REGULAR or CO or AMO)
