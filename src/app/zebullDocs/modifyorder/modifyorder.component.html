Modify Order

URL

< BASE URL > + v1/placeOrder/modifyOrder


Method

Post

Input Parameters

S.NO KEY VALUE
1 discqty Int Discqty
2 exch String Exchange
3 trading_symbol String Trading Symbol
4 nestOrderNumber Int Nest Order Number
5 prctyp String Price Type
6 price Int Price
7 qty Int Quantity
8 trading_symbol String Id_Symbol
9 trigPrice Int Trigger Price
10 filledQuantity Int Filled Quantity

        {{'{'}}
             "discqty": "0",

             "trigPrice":"00.00",

       {{'}'}}

Success Response

S.NO PARAMETERS DESCRIPTION
1 stat String Ok
2 Result Int Nest order number

Error

S.NO PARAMETERS DESCRIPTION
1 stat String Not_Ok
2 emsg
  • Session Expired

  • Not able to Retrieve ModifyOrder

  • Error Occurred : < reason>

  • You have to enter Date dd-MM-yyyy for GTD Orders.

  • Invalid Good Till Date: Cannot be in the Past.

  • Invalid Order Status

  • MARKET PROTECTION is not allowed for Limit Order Type

  • No Data
