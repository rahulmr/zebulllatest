Add Scrips

URL

< BASE URL> + v1/marketWatch/addScripToMW


Method

Post

Input Parameters

S.NO KEY VALUE
1 mwName String Market Watch Name
2 exch String Exchange NSE
3 symbol String Symbol Token

Example

      {{'{'}} 
           "mwName": "abc",
           "exch": "NSE",
           "symbol": "46801"
       {{'}'}}


Success Response

S.NO PARAMETERS DESCRIPTION
1 stat String Ok
2 Result String success

S.NO PARAMETERS DESCRIPTION

  • BAD_INPUT
