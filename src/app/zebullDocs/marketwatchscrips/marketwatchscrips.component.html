Market Watch Scrips

URL

< BASE URL > + v1/marketWatch/fetchMWScrips


Method

Post

Input Parameters

S.NO KEY VALUE
1 mwName String Market Watch Name

      {{'}'}}

Success Response

S.NO PARAMETERS DESCRIPTION

3 indexvalues String JSON array with indices name and values
4 ScripCount int No of scrips in market watch

2 emsg String
